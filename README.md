# Строительный сайт v2 — GitHub Pages / WordPress-ready

## Что добавлено в v2
1. Фото-заглушка собственника и усиленный блок личной ответственности.
2. Расширенные кейсы с площадью, сроками и составом работ.
3. Блок фото до/после под будущие реальные материалы.
4. Фиксированные кнопки WhatsApp, Telegram и звонка.
5. SEO-текст на главной под строительную компанию в Челябинске.
6. Квиз-калькулятор предварительной стоимости.
7. Блок видео/отзывов/документов.
8. Блок под договор, реквизиты и гарантийные обязательства.
9. Интерактивная карта объектов как дизайн-заготовка.
10. Улучшенные карточки услуг с показом популярных направлений.
11. Более сильный hero с конкретными УТП.
12. 3D SVG-иконки и изображения-заглушки по тематике строительства.
13. Микроразметка LocalBusiness и Service.

## Структура
- `index.html` — главная страница.
- `services/` — 22 SEO-страницы услуг.
- `assets/css/style.css` — стили.
- `assets/js/main.js` — меню, модалки, демо-формы, раскрытие услуг.
- `assets/img/` — SVG-изображения и иконки-заглушки.

## Для выгрузки на GitHub Pages
Загрузить содержимое папки `stroy-company-site-v2` в репозиторий. В настройках GitHub Pages выбрать ветку `main` и корневую папку `/root`.

## Что заменить перед запуском
- Номер телефона, WhatsApp, Telegram.
- Фото собственника, сотрудников и объектов.
- Реальные кейсы, отзывы, документы.
- Подключить формы к WordPress/CF7/Fluent Forms, Telegram/CRM и Яндекс Метрике.
- Заменить демо-карту на Яндекс.Карты.
