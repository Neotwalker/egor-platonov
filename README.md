# СтройФорма — сайт строительной компании

Статическая версия сайта для выгрузки на GitHub Pages.

## Структура
- `index.html` — главная страница
- `services/*.html` — SEO-страницы услуг
- `assets/css/style.css` — стили
- `assets/js/main.js` — меню, модалки, формы-заглушки
- `assets/img/*.svg` — временные иллюстрации-заглушки

## Что заменить перед продакшеном
1. Логотип, телефон, WhatsApp/Telegram.
2. Реальные фото собственника, объектов, до/после и отзывы.
3. Подключить обработчик форм: WordPress/Contact Form 7/CRM/Telegram.
4. Добавить Яндекс Метрику и цели.
5. Подключить SSL на хостинге.

## WordPress-интеграция
Верстка подготовлена блоками. Для WP лучше сделать ACF-поля под hero, услуги, кейсы, галерею, FAQ и формы.
