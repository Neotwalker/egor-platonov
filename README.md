# Egor Platonov — site v3

Статическая версия сайта для GitHub Pages.

## Что внутри
- главная страница с интерактивным hero, квизом, картой, кейсами и FAQ;
- архив услуг `/services/index.html`;
- 22 SEO-страницы услуг;
- политика конфиденциальности;
- адаптивная вёрстка, vanilla JS, Leaflet-карта.

## Публикация на GitHub Pages
1. Создайте репозиторий.
2. Загрузите содержимое архива в корень репозитория.
3. В Settings → Pages включите публикацию из ветки `main` / корня.
4. Дождитесь публикации сайта.
