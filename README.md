# Egor Platonov — демонстрационная версия сайта

Статическая вёрстка сайта строительной компании для показа клиенту.

## Что внутри
- главная страница;
- каталог услуг;
- 22 страницы услуг;
- 7 страниц категорий услуг;
- страница отзывов;
- политика конфиденциальности;
- адаптивная вёрстка для десктопа, планшетов и мобильных;
- модальные формы, квиз, FAQ, кейсы, блок до/после, карта и мобильная CTA-панель.

## Как открыть
Откройте `index.html` в браузере или загрузите содержимое архива в репозиторий GitHub Pages.

## Что заменить перед запуском
- телефон, почту и ссылки на мессенджеры;
- реальные фото собственника и объектов;
- реальные отзывы и ссылки на площадки;
- реквизиты и юридические данные;
- реальные цены, сроки, кейсы и документы.
