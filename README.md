# Эксперт Сервис — демонстрационная версия сайта

Статическая адаптивная вёрстка сайта строительной компании «Эксперт Сервис» для согласования дизайна и дальнейшей интеграции в WordPress.

## Состав проекта

- главная страница;
- каталог услуг;
- 22 страницы услуг;
- 7 страниц категорий;
- страницы видео, отзывов, FAQ и политики конфиденциальности;
- страница 404;
- кейсы, сравнение «до/после», формы, модальные окна и мобильные быстрые действия;
- локальные библиотеки и шрифт без CDN.

## Структура ресурсов

```text
assets/
├── css/site.css
├── js/main.js
├── img/
└── libs/
    ├── inter/
    │   ├── inter.css
    │   └── files/
    ├── swiper/
    │   ├── swiper-bundle.min.css
    │   └── swiper-bundle.min.js
    └── fancybox/
        ├── fancybox.css
        └── fancybox.umd.js
```

## Запуск

Откройте `index.html` или запустите локальный сервер:

```bash
python -m http.server 8000
```

Адрес:

```text
http://localhost:8000/
```

Содержимое папки проекта можно загрузить в корень репозитория GitHub Pages.
